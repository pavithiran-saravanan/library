# Library
A responsive library app to add and keep track of books

# Demo
Check out the live demo of the project here https://pavithiran-saravanan.github.io/library/

# Built With
- HTML
- CSS
- Javascript

# Features
- Add/Remove books
- Mark books as read/unread
- Responsive design
